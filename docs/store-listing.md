# Eureka Chrome Web Store Listing Draft

## Name

Eureka Research Copilot

## Short Description

Turn any web page into a cited research report with a local multi-agent AI workflow.

## Detailed Description

Eureka helps students, researchers, and developers read long web pages without losing context. Click the extension, ask a question, and Eureka sends the current page text to your configured local backend. The backend runs a multi-agent workflow that can summarize the page, plan a deeper search, collect evidence, verify citations, and generate a structured Markdown report.

Key features:

- Read the current page only after user action.
- Summarize pages or run deeper research.
- Show Agent Trace for transparent execution.
- Support Qwen, Kimi, DeepSeek, Zhipu GLM, OpenAI, and custom OpenAI-compatible models through the local backend.
- Export saved reports as Markdown or PDF from the extension or local Web Demo.
- Store settings locally with Chrome storage.

## Privacy Summary

Eureka does not collect cookies, passwords, browsing history, or credentials. The extension reads the active tab only after the user clicks the research button. Page text is sent to the user-configured local backend endpoint.
